Cara Pakai

YouTube Live Chat OBS

1

Buka OBS Studio

Tambahkan Browser Source baru. URL YouTube live chat:
https://youtube.com/live_chat?is_popout=1&v=VIDEO_ID

2

Atur Ukuran

Set Width: 400 dan Height: 600.

3

Copy CSS

Klik Salin CSS lalu paste ke kolom Custom CSS di OBS Browser Source.

4

Aktifkan Transparansi

Centang "Shutdown source when not visible". Background overlay sudah transparan otomatis.

5

Atur Posisi di Scene

Seret Browser Source ke posisi yang kamu inginkan. Chat overlay siap live!

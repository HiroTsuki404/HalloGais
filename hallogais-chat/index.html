Cara Pakai

YouTube Live Chat OBS

1

Buka OBS Studio

Tambahkan Browser Source baru. Masukkan URL YouTube live chat:
https://youtube.com/live_chat?is_popout=1&v=VIDEO_ID
Ganti VIDEO_ID dengan ID video stream kamu.

2

Atur Ukuran

Set Width: 400 dan Height: 600 (atau sesuai kebutuhan).

3

Copy CSS

Klik tombol Salin CSS di bawah ini lalu paste ke kolom Custom CSS di pengaturan Browser Source OBS.

4

Aktifkan Transparansi

Centang opsi "Shutdown source when not visible" dan "Refresh browser when scene becomes active".

5

Atur Posisi di Scene

Seret Browser Source ke pojok layar yang kamu inginkan. Chat overlay siap tampil saat live!

Paste ke Custom CSS di OBS Browser Source.

⚙️ Cara Pasang di OBS
Selesai dalam kurang dari 1 menit.
1

Generate link overlay kamu

Scroll ke bawah ke bagian Link Generator, isi nama channel dan pilih tema, salin link yang muncul.

2

Buka OBS Studio

Di panel Sources, klik tombol + → pilih Browser Source.

3

Paste link overlay

Tempel link tadi ke kolom URL. Set ukuran: Width: 400  Height: 600. Centang Shutdown source when not visible.

4

Aktifkan Chroma Key / Transparan

Klik kanan Browser Source → Filters → tambah Color Correction atau gunakan filter Chroma Key jika perlu. Background overlay sudah transparan otomatis.

5

Hubungkan ke StreamElements (opsional)

Untuk menampilkan chat live sungguhan dari YouTube/Twitch, gunakan StreamElements Widget dan tempel CSS dari seksi di bawah ke editor custom CSS-nya.

🎨 CSS untuk StreamElements
Pilih tema → salin CSS → paste ke StreamElements Chat Widget → simpan.
